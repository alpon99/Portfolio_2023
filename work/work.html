NCS Process

  • UI/UX Renewal Project

    UI/UX 리뉴얼 기획서입니다.
    인하공업전문대학 홈페이지의 UI/UX를 개선사항, 리뉴얼 계획입니다.

  • UI/UX Implementation

    UI/UX 구현 기획서입니다.
    디자인 컨셉, 기술적인 구현 방법 등 구체적인 리뉴얼 계획입니다.

  • UI/UX Usability Testing

    UI/UX 사용성 테스트입니다.
    디자인과 사용성을 검증하고, 사용자 피드백을 수집하여 이를 개선합니다.

  • UI/UX Concept Plannig

    UI/UX 콘셉트 기획입니다.
    포트폴리오의 목표를 정하고, 핵심 컨셉과 디자인 시스템을 확립합니다.

  • UI Architecture Design

    UI 아키텍쳐 설계입니다.
    정보구조도, 플로우차트 등 웹의 전체적인 구조와 흐름을 계획합니다.

  • UI/UX Analysis

    UI/UX 요구 분석입니다.
    사용자의 요구사항을 수집하고 사용자의 방문 목적, 니즈 등을 분석합니다.

  • Problem-Solving

    문제해결 시나리오입니다.
    포트폴리오 코딩 중 발생한 문제를 식별하고 해결하는 과정입니다.

  • Div Environment Setup

    개발자 환경구축입니다.
    원활한 개발을 위해 작업환경을 설정하는 과정으로 깃허브를 다룹니다.
